# styles

## Purpose
Theme preset stylesheets and third-party icon CSS. The base Tailwind layer and CSS variables live in `src/app/globals.css`; this folder holds swappable color presets and flag icons.

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `presets/` | Generated color-preset CSS (CSS-variable overrides per theme) |
| `flag-icons/` | Vendored flag-icon CSS (country flags) |

### `presets/`
| File | Description |
|------|-------------|
| `tangerine.css` | "Tangerine" color preset |
| `brutalist.css` | "Neo Brutalism" color preset |
| `soft-pop.css` | "Soft Pop" color preset |

The default uses the **shadcn neutral** theme (defined in `globals.css`); the files above are additional presets inspired by Tweakcn.

## For AI Agents

### Working In This Directory
- Preset CSS in `presets/` is **generated** by `src/scripts/generate-theme-presets.ts` — edit the generator/source, then run `npm run generate:presets`, rather than hand-editing the output.
- Each preset overrides CSS custom properties (tokens) for both light and dark modes; follow the existing structure when adding a new preset and register it in `src/lib/preferences/`.

## Dependencies

### Internal
- Generated by `src/scripts/`; selected via `src/lib/preferences/` + `stores/preferences/`.
