# server

## Purpose
Next.js **Server Actions** — server-side functions callable from client components/forms.

## Key Files
| File | Description |
|------|-------------|
| `server-actions.ts` | `"use server"` actions (e.g. setting preference cookies server-side, form submission stubs) |

## For AI Agents

### Working In This Directory
- Files here run on the server. Keep them annotated with `"use server"` and never import client-only modules (`*.client.ts`) into them.
- This template has no real backend; actions are thin stubs/cookie writers. Add real persistence here when integrating a backend.
