# (external)

## Purpose
Route group for public, non-app-shell pages (e.g. the landing/root page). Routes here render outside the dashboard chrome. The group name `(external)` does not appear in the URL.

## Key Files
| File | Description |
|------|-------------|
| `page.tsx` | The site root (`/`) page |

## For AI Agents

### Working In This Directory
- Add landing/marketing/public routes here, not under `(main)`.
- These pages should not depend on the dashboard sidebar/layout or authenticated providers.
